# Notas

- `.exe` roda via Wine/Winetricks/Lutris, não nativamente como Windows NT.
- Sons e jogos originais do Windows 7 são proprietários, por isso foram usados jogos livres.
- Microsoft Store real não pode ser incluída; foi substituída por Nexus Store.
- Tethering USB depende do celular expor interface USB Ethernet/RNDIS/Modem e do kernel reconhecer o dispositivo.
